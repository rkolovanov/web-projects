Абзац. Тэг P - определяет текстовый абзац. Тег < p> является блочным элементом, всегда начинается с новой строки, абзацы текста идущие друг за другом разделяются между собой отбивкой. Величиной отбивки можно управлять с помощью стилей. Если закрывающего тега нет, считается, что конец абзаца совпадает с началом следующего блочного элемента.

Покажем разные стили: жирный; жирный, только с помощью strong; курсив; курсив, только с помощью em; перечеркивание; подчеркивание;
